ยงmetadata-gen

metadata-gen is a library for extracting and processing metadata for static site generators. It supports various formats including YAML, TOML, and JSON front matter in Markdown files.

This crate provides functionality to:

  • Extract metadata from content files
  • Process and validate metadata
  • Generate keywords based on metadata
  • Create meta tags for HTML documents
  • Asynchronously extract metadata from files
